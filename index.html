// 01 — About

Precision. Innovation. Impact.

I'm Evance Ochiel — a GIS Developer and Spatial Analyst holding a BSc. in Geospatial Information Science with IT (Maseno University). Over 7 years, I've translated complex spatial data into intelligent, production-ready systems used by governments, private firms, and everyday commuters.

Currently a Geospatial Data Management Officer at Kenya's Directorate of Resource Surveys and Remote Sensing, I own the full lifecycle of geospatial projects — from satellite imagery ingestion and GPS field collection to predictive modeling, automation, and interactive dashboards.

On the side, I ship Android apps, integrate REST APIs, manage cloud-hosted spatial databases, and apply machine learning to geospatial challenges — because the best solution is often the one that hasn't been built yet.

// 02 — Capabilities

What I Build

A multidisciplinary engineering toolkit — spanning satellite imagery to mobile apps, databases to AI — built for real-world deployment.

GIS Analysis & Mapping

Full-cycle spatial analysis — buffering, overlay, interpolation, proximity — using ArcGIS Pro, QGIS, ERDAS Imagine, and Google Earth Engine. Professional cartographic output for any scale.

Android App Development

Native Android apps in Kotlin/Java with Maps SDK, GPS integration, offline-first architecture, Firebase backend, and full Play Store deployment — from concept to production.

Web & WebGIS Development

Interactive web mapping with Leaflet.js, Mapbox GL, OpenLayers, and ArcGIS JS API. Full-stack web apps using HTML5, CSS3, JavaScript, and PHP with spatial backends.

Spatial Database Management

PostgreSQL + PostGIS for complex spatial queries, indexing, and geospatial functions. MySQL, SQLite, and Firebase for mobile/web data pipelines. Schema design and performance tuning.

Automation & Data Pipelines

Python-driven geoprocessing scripts, scheduled ETL pipelines, data cleaning workflows, format converters (GeoJSON, Shapefile, GeoPackage), and ArcPy / GDAL automation at scale.

API Integration & Services

REST API design and consumption — Google Maps Platform, OpenStreetMap, GTFS feeds, Firebase, Overpass API, and custom spatial microservices. JSON/GeoJSON data flows.

Spatial Data Visualization

Dynamic dashboards, ArcGIS StoryMaps, Tableau spatial charts, and custom D3.js / Recharts visualizations. Turning raw geospatial data into compelling, decision-ready narratives.

AI & Predictive Geospatial Analytics

Machine learning applied to spatial data — land cover classification, route optimization, predictive urban modeling, and AI-moderated community features. Python ML stack + GEE.
